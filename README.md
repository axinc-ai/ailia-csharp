# ailia-csharp

This is a sample to execute ailia's Predict API using C#.

## Requirements

- Windows 10
- VisualStudio 2019

## Build

- Open `ailia-csharp.sln`.
- Change active platform `Any CPU` to `x64`.
- Build.

## Run

- Please place ailia.dll to `/ailia-csharp/ailia-csharp/bin/x64/Debug`.
- Run.
